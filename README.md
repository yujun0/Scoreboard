## Quick Start

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yujun0/Scoreboard.git
cd Scoreboard

# Install dependencies
npm install

# Start development server
npm run dev
```

The app will be available at `http://localhost:5173/`

## Usage

1. **Set Team Names** - Click on "Team A" or "Team B" to edit team names
2. **Add Points** - Use the +2, +3, +1 buttons to add points
3. **Correction** - Use the Correction button to subtract 1 point if needed
4. **Timer** - Click the timer to set custom time, then use Start/Pause to control it
5. **Reset** - Click "Reset Game" to start a new game

## Build & Deploy

```bash
# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```
